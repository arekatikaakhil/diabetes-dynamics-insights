Data Cleaning and Preprocessing Source Code

This folder contains Python scripts and notebooks used for data cleaning, exploratory data analysis (EDA), and data quality assessment as part of the Diabetes Dynamics project.

File Structure:

- Cleaning script: Contains Python scripts for cleaning and preprocessing raw data.
- EDA Script: Contains Python scripts for exploratory data analysis.
- Data Quality Assessment: Contains scripts  for assessing the quality of the dataset.
- Cleaned Dataset: cleaned dataset after preprocessing.
